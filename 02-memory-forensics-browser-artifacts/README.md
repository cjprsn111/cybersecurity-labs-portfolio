# Memory Forensics and Browser Artifact Analysis

## Objective

Analyze Windows memory and browser artifacts to identify user activity, running processes, and possible suspicious behavior.

## Tools Used

- Volatility
- DB Browser for SQLite
- Linux command line

## Summary

In this lab, I reviewed memory forensic artifacts and extracted browser history data. I analyzed process listings, process trees, network connections, and Chrome history records to help build an investigation timeline.

## Key Findings

- Reviewed running processes using Volatility
- Analyzed process parent-child relationships
- Reviewed network connection artifacts
- Opened Chrome history data in DB Browser for SQLite
- Reviewed URLs, page titles, visit counts, and timestamp fields

- Memory forensics
- Browser artifact analysis
- SQLite database review
