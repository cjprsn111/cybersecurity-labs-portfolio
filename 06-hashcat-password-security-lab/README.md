# Password Hash Cracking Lab with Hashcat

## Objective

Use Hashcat to crack an MD5 hash in an authorized lab environment and understand weak password risks.

## Tools Used

- Hashcat
- Kali Linux
- RockYou wordlist
- Hashcat rules

## Summary

In this lab, I identified an MD5 hash and used Hashcat with a dictionary-based attack to recover the plaintext password. This lab demonstrated how weak passwords can be recovered and why strong password practices are important.

## Skills Practiced

- Hash identification
- Password security testing
- Dictionary attacks
- Hashcat command usage
- Linux command line
- Security documentation
