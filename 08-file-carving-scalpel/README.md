# File Carving and Data Recovery with Scalpel

## Objective

Use Scalpel to recover files from a forensic disk image.

## Tools Used

- Scalpel
- Kali Linux
- Linux command line

## Summary

In this lab, I used Scalpel to carve a forensic disk image and recover multiple file types. The recovered output included folders for PDFs, Excel files, PNGs, JPGs, and GIFs. I reviewed the carved output folders and documented the recovery process.

## Key Findings

- Recovered multiple file types from a forensic disk image
- Reviewed carved output folders
- Identified recovered Excel, PDF, PNG, JPG, and GIF files
- Practiced evidence recovery documentation

- File carving
- Digital forensics
- Data recovery
- Evidence analysis
- Linux command line
