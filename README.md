# Cybersecurity Labs Portfolio

This repository contains cybersecurity training labs completed in a controlled and authorized lab environment. These projects focus on network analysis, vulnerability assessment, digital forensics, password security, wireless reconnaissance, and ethical exploitation.

## Projects

1. Packet Capture Analysis - FTP Data Exfiltration Investigation
2. Memory Forensics and Browser Artifact Analysis
3. Wireless Network Reconnaissance Lab
4. OpenVAS Vulnerability Assessment
5. Nmap Port Scan and Service Enumeration
6. Password Hash Cracking Lab with Hashcat
7. Metasploit FTP Vulnerability Validation
8. File Carving and Data Recovery with Scalpel

## Tools Used

- Kali Linux
- Wireshark
- Nmap
- OpenVAS / Greenbone
- Volatility
- DB Browser for SQLite
- Hashcat
- Metasploit
- Scalpel
- Linux command line

## Disclaimer

All labs were completed in a controlled, authorized training environment. No testing was performed against unauthorized systems or networks.
