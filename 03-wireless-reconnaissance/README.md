# Wireless Network Reconnaissance Lab

## Objective

Perform wireless reconnaissance in a controlled lab environment to identify nearby access points and analyze Wi-Fi network characteristics.

## Tools Used

- Kali Linux
- Wireless monitoring tools
- Linux command line

## Summary

In this lab, I reviewed wireless scan output to identify access points, signal strength, channels, encryption types, ciphers, authentication methods, beacon activity, and station information.

## Key Findings

- Identified wireless access points
- Reviewed BSSID and ESSID information
- Analyzed signal strength and channel usage
- Compared encryption and authentication methods
- Reviewed station/client activity

## Skills Practiced

- Wireless reconnaissance
- Wi-Fi security analysis
- 802.11 network review
- Kali Linux
- Security documentation
