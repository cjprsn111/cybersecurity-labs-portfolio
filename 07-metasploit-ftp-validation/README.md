# Metasploit FTP Vulnerability Validation Lab

## Objective

Use Metasploit in a controlled lab environment to validate a known FTP vulnerability on a Metasploitable virtual machine.

## Tools Used

- Metasploit
- Kali Linux
- Nmap
- Metasploitable

## Summary

In this lab, I identified a vulnerable FTP service running vsftpd 2.3.4 on a Metasploitable VM. I selected the matching Metasploit module, configured the target options, and validated the vulnerability in an authorized training environment.

## Key Findings

- Identified vsftpd 2.3.4 through service enumeration
- Selected the matching Metasploit module
- Configured target host, port, payload, and listener options
- Validated the vulnerability in a controlled lab

## Skills Practiced

- Ethical exploitation
- Vulnerability validation
- Metasploit module usage
- Nmap service enumeration
- Penetration testing documentation

## Disclaimer

This lab was performed only against an intentionally vulnerable virtual machine in an authorized lab environment.
